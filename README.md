# millennium-dawn-2026
Millennium Dawn 2026 Overhaul
For HOI4 1.14.1 · Based on Millennium Dawn: Modern Day Mod
The last major Millennium Dawn update removed the 2017 start date, keeping only the 2000 one. This mod restores a modern start — set to January 1, 2026.
What's Included
Conflict Updates
All major active conflicts updated as of January 2026, including Yemen, Syria, Sudan, Libya, and Somalia.
Political Overhaul
Ruling coalitions, party compositions, and leadership updated to reflect early 2026 across multiple European countries, with special focus on India and the USA. An ongoing political parties overhaul for the broader world was in progress at the time of archiving.
Status
Archived. Fully playable in its current state but incomplete — not every country has received a political update. No further development is planned.
Installation

Open the Paradox Launcher and create a new mod — name it anything you like
Click Show in Folder on the newly created mod
Download and unzip the mod files
Paste the unzipped contents directly into that folder
Launch the mod from the Paradox Launcher

Compatibility

HOI4 version: 1.14.1
Requires Millennium Dawn: Modern Day Mod

Notes
This mod was developed as an ongoing project and archived before completion. It is distributed as-is with no guarantees of balance or completeness beyond what is described above.
